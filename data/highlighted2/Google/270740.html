loyal to Windows China banned Microsofts NASDAQ MSFT Windows for governmentrelated procurement last May The ban was part of a state directive over energysaving measures It was definitely an angry reaction against Microsofts decision to end support for Windows XP The old Windows XP OS was still used by more than of all computers in China Microsofts discontinuation of supportupdates for Windows XP clearly exposed Chinese computers to virus and hacking attacks The Chinese government did not like this and made its displeasure felt Microsoft was previously in danger last year from Chinas antitrust allegations over the Windows operating system Chinas government even helped create its own homegrown OS NeoKylin that it wanted as a replacement for Windows The antiWindows sentiment of However a China governmentdistributed Linux OS with the aesthetics of Windows XP could potentially reach million PC users within a few years Investors should therefore feel relieved that Nadella is now doing his best to placate the irate Chinese In a marked departure from Microsofts grand Universal Windows design Chinese government institutions will get a special custom Windows OS This deal with Chinas government is different from the distribution deal with Baidu NASDAQ BIDU Microsoft only modified the Windows for Baidus distribution in exchange for using Baidus search engine instead of Bing
