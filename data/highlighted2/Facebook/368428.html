Users can access these chatbots by typing in a business name in the search bar of the Messenger application or through searching for a business on Facebook and hitting the message icon on the businesss Facebook page This will bring the user to a Messenger chat with the business After the user sends the first message the chatbot will reply almost immediately The user and chatbot then enter a dialogue As they stand the chatbots are actually quite complex The bots can send text structured messages images links and calltoaction buttons Users however cannot yet make transactions directly through Messenger The Messenger platform currently does not allow payments directly through a credit card added to Messenger so users are required to redirect to the web to pay for an item To make sure users retain control each chatbot has a block button so the user can silence the chatbot by simply selecting that button Facebook has also made it relatively easy for merchants to have their own chatbot Merchants can either build their own bots or work with Facebooks Bot Engine Bot Engine built on Facebooks acquisition of natural language interface startup Witai makes creating a chatbot relatively easy for all merchants regardless of size or technical savvy All merchants are required to do is give the Bot Engine sample conversations and the Bot Engine will learn to handle similar conversations with users on its own There are currently three featured chatbots on Facebooks website Those include Spring Poncho and CNN We have used all three and will explain each chatbot briefly here We do recommend however all our readers try these chatbots for themselves to get a better understanding of the secular transition at work here Spring acts as a personal shopping assistant on Messenger At first the Spring chatbot prompts the user if they are interested in mens or womens clothing The user then enters a dialogue with the Spring chatbot wherein the chatbot will recommend certain articles of clothing to the user based on what the user asks for For example the photo below details an individual who is interested in shoes priced in the to range The individual messages this interest to Spring and the Spring chatbot replies with recommendations of shoes in the to price band Poncho meanwhile provides users with weather updates via Messenger Users can prompt Poncho with questions regarding the current and forecast weather in certain places and Poncho will reply with a text or image answering the users prompt Users can also set automatic weather update times when Poncho will message the user relevant weather updates at a certain time every day The most wellknown of the featured Messenger chatbots is the CNN chatbot The CNN chatbot is designed to deliver the latest headlines and curated news coverage to the user The chatbot is set up to deliver top headlines to the user on a daily basis as well as to provide suggested stories based on the users input and activity What It All Means For The Future Of Facebook From the three featured chatbots alone a user can message a friend order a pair of shoes check the fiveday weather forecast and catch the latest headlines all through the Messenger application This is on top of users already being able to order an Uber from the Messenger app We believe this is only the tip of the iceberg in terms of what Facebook is attempting here which is essentially creating a consolidated communication forum connecting all merchants businesses with Facebook pages to all consumers Facebook users In its initial
